# Code Auxiliary（コード補助）

Claude Code が何をしているかをリアルタイムで解説してくれる Web アプリです。
別のターミナルで Claude Code を使いながらこのアプリを開いておくと、ファイル操作・コマンド・プロンプト・Claude の返答をすべてキャプチャして、わかりやすく解説します。

---

## はじめる前に確認すること

| 必要なもの | 確認方法 |
|---|---|
| **Node.js 18 以上** | `node -v` |
| **Claude Code** | `claude --version` |
| **このアプリのフォルダ** | `~/code-explainer/` に置いてください |

---

## セットアップ手順

### STEP 1 ― パッケージをインストールする

```bash
cd ~/code-explainer
npm install
```

### STEP 2 ― サーバーを起動する

```bash
npm run dev
```

ターミナルに `Server running at http://localhost:3001` と表示されたら起動成功です。
**このターミナルは起動したままにしておいてください。**

### STEP 3 ― ブラウザで開く

別のターミナルまたはブラウザで以下にアクセスします。

```
http://localhost:3001
```

### STEP 4 ― アプリ内でセットアップする

ブラウザを開いたら、左メニューの **「ガイド」** タブ →「はじめてのセットアップ」を開きます。

#### 4-a. Claude Code のフックを設定する

「フックを自動で設定する」ボタンを押してください。
`~/.claude/settings.json` に自動で設定が書き込まれ、Claude Code とこのアプリが連携されます。

> **フックとは？**
> Claude Code が何か操作するたびにこのアプリへ自動通知するための仕組みです。
> これを設定しないと、ログやプロンプトがアプリに表示されません。

### STEP 5 ― 動作確認する

別のターミナルで Claude Code を起動して、何か指示を送ってみてください。

```bash
# 例：別のターミナルで
claude
```

Claude Code に話しかけると、このアプリのタブにログ・プロンプト・Claude の返答がリアルタイムで表示されれば**連携成功**です。

## 各タブの説明

| タブ | 説明 |
|---|---|
| **全体の概要** | すべての操作をまとめて AI が一括解説。まず何をしているか把握したいときに使う |
| **ログ一覧** | ファイル作成・編集・コマンド実行の記録。コードの中身も確認できる |
| **プロンプト** | Claude Code に送ったお願い文の一覧。上部の「次のアクション提案」が便利 |
| **履歴** | 30 分ごとに区切ったトーク単位で過去の操作を振り返れる |
| **ガイド** | このアプリの使い方・セットアップ・FAQ |
| **アップグレード** | プラン変更・招待コードの入力 |

## トラブルシューティング

### 「停止中」のまま動かない

サーバーが起動しているか確認してください。

```bash
cd ~/code-explainer && npm run dev
```

ポート 3001 が別のアプリに使われていると起動しません。その場合は `server/api.ts` の `PORT` を変更してください。

### Claude Code を使ってもログが表示されない

フックが設定されていない可能性があります。
「ガイド」→「はじめてのセットアップ」の STEP 2「フックを自動で設定する」を実行してください。
設定後、**Claude Code を一度再起動**してください。

### 「API キーが設定されていません」と出る

無料プランの場合、API キーが必要です。
「ガイド」→「はじめてのセットアップ」の STEP 1 でキーを設定してください。
Pro / MAX プランの場合は、アップグレードタブで招待コード or Stripe 決済を完了しているか確認してください。

### ログが貯まりすぎた

サイドバー下部の「すべてクリア」ボタンで全データを削除できます（元に戻せません）。

## ディレクトリ構成

```
code-explainer/
├── hooks/
│   ├── log-action.js      # ファイル操作をキャプチャ（PostToolUse フック）
│   ├── log-prompt.js      # プロンプトをキャプチャ（UserPromptSubmit フック）
│   └── log-assistant.js   # Claude の返答をキャプチャ（Stop フック）
├── server/
│   └── api.ts             # Express サーバー（SSE・AI 呼び出し・フック設定 API）
├── src/
│   ├── components/        # UI コンポーネント
│   ├── hooks/             # カスタム React フック
│   └── types/             # TypeScript 型定義
├── .env                   # API キー（git 管理しない）
└── package.json
```

データは `~/.code-explainer/` に保存されます。

```
~/.code-explainer/
├── logs/              # ファイル操作ログ（JSON）
├── prompts/           # プロンプトログ
├── assistant-messages/ # Claude の返答ログ
├── history/           # 履歴データ
└── sessions.json      # Pro / MAX セッショントークン
```
