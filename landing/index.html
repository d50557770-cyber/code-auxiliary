Setup

3 ステップで始める

1

ターミナルに以下のコードを貼り付ける

git clone https://github.com/d50557770-cyber/code-auxiliary
cd code-auxiliary && npm install
npm run dev

⚠️ Windows の方は WSL が必要です。Mac / Linux はそのまま動きます。

2

ブラウザでフック設定

localhost:3001 を開き、ガイドタブ →「フックを自動で設定する」ボタンを押すだけ。

3

Claude Code に話しかける

プライバシーポリシー

最終更新日：2026年2月24日　／　事業者：Taiga

1. 収集する情報

Code Auxiliary は、Claude Code のログ（ファイル操作・コマンド実行・プロンプト内容）をローカルで読み取り、AI 解説のためにサーバーへ送信します。個人を特定できる情報は収集しません。

2. 情報の使用目的

収集した情報は、AI による操作解説の生成のみに使用します。第三者へ販売・提供することはありません。

3. データの保存

ログデータはお使いのマシン上（~/.code-explainer/）に保存されます。サーバー側には解説生成に必要な最小限のデータのみを一時的に処理し、保存しません。

4. セキュリティ

API エンドポイントはレート制限・認証・セキュリティヘッダーで保護されています。通信は HTTPS で暗号化されます。

5. お問い合わせ

プライバシーに関するご質問は d50557770@gmail.com までご連絡ください。
