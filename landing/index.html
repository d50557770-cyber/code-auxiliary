Setup

3 ステップで始める

1

リポジトリをクローンして起動

$ git clone https://github.com/d50557770-cyber/code-auxiliary

$ cd code-auxiliary && npm install

$ npm run dev

2

ブラウザでフック設定

localhost:3001 を開き、ガイドタブ →「フックを自動で設定する」ボタンを押すだけ。

3

Claude Code に話しかける

別のターミナルで Claude Code を使うと、操作がリアルタイムで表示されます。
